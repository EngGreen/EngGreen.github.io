『게임 프로그래머 입문 올인원』 0. OT 및 세팅

게임 프로그래머 입문 올인원 by rookiss

  • 0. OT 및 세팅
  • 1-1. 프로그래밍 개론 및 변수와 정수
  • 1-2. 산술 연산, 비교 연산과 논리 연산
  • 1-3. 분기문 및 반복문

※ 강의 정보

[게임 프로그래머 입문 올인원] C++ & 자료구조/알고리즘 & STL & 게임 수학 & Windows API & 게임 서버

어디부터 시작할지 막막한 게임 프로그래밍 입문자를 위한 All-In-One 커리큘럼입니다. C++, 자료구조/알고리즘, STL, 게임 수학, Windows API, 게임 서버 입문으로 이어지는 알찬 커리큘럼으로 게임 프로그래밍 기초를 폭넓게 공부합니다.

강의 선택 이유

  • 굉장히 만족스러웠던 게임 프로그래머 취업 전략 가이드 (by Rookiss)강의.
  • 책으로 공부하는 동안, 흥미를 잃지 않고 전체적인 개발과정을 겪어볼 수 있음.
강의 커리큘럼

C++ → 자료구조 알고리즘 → STL → 게임수학 → Windows API → 게임서버 입문

  1. Windows를 이용한 포트리스 모작 (게임수학에 대한 내용 학습 가능)
  2. 2D 로그라이크 제작 → 온라인 서버 연동
Visual Studio 2022 세팅
  1. 마이크로소프트 visual studio 홈페이지 접속
  2. 커뮤니티 → 무료 다운로드 버튼으로 셋업파일 다운 후 실행
  3. 실행 후, C++를 사용한 데스크톱 개발 및 C++를 사용한 게임 개발 체크 후 설치 진행
  4. 실행 후 새 프로젝트 만들기
  5. 언어 C++, 프로젝트 형식 콘솔 설정, 콘솔 선택 후 다음
  6. 원하는 폴더 위치에 프로젝트 생성
  7. 위쪽 메뉴의 디버그 → 디버깅 혹은 F5(디버깅 단축키) 사용 시 다음과 같은 화면 출력
  8. 준비 완료!

※ visual assist 란?

Visual assist는 Visual studio를 사용하여 코딩할 때, 도움을 주는 편리한 애드온이다. 1년에 $279정도의 가격으로 제공되며, 한 달 동안 무료버전으로 사용이 가능하다. 대부분의 게임회사에서 프로그래머에게 구매해준다고 한다.
